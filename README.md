# Command-Line Mode

Command-Line Mode is a Jump King practice mod that adds a small Vim-like command prompt.

Current feature:

- Press `:` to open command-line mode
- Type a screen number from `1` to `169`
- Press `Enter` to teleport to that screen
- Invalid input and out-of-range screen numbers are ignored

This mod is intended as a practice tool, not for real-time runs or leaderboard submissions.

## Commands

| Command | Action |
| --- | --- |
| `:1` | Teleport to screen 1. |
| `:100` | Teleport to screen 100. |

## Keys

| Key | Action |
| --- | --- |
| `:` | Open command-line mode. |
| `0` to `9` | Enter a screen number. |
| `Backspace` | Delete one digit. |
| `Enter` | Run the command. |
| `Esc` | Close command-line mode. |
